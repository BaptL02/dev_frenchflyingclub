INSTALLATION VMR

Ce petit document a pour but d'expliquer le fonctionnement et l'installation de fichier VMR dans le client vPilot pour Microsoft Flight Simulator.

Pour commencer, il faut avoir vPilot installé, mais aussi les livrées que l'on souhaite afficher en jeu présentes dans le fichier Community.

Les livrées du club sont disponibles dans la section "Nos livrées", nous recommandons à tous les membres d'installer l'A320Neo de FlyByWire ainsi que celle du DA62, celles-ci servant de modèle par défaut pour tous les avions du club.

FRENCH FLYING CLUB

Afin de voir les autres avions du French Flying Club sous leur meilleur angle vous devrez suivre les étapes suivantes :

Commencez par télécharger dans la section "Nos livrées" les livrées pour le/les avion(s) que vous possédez ainsi que le fichier .vmr correspondant.

1- Dans le client vPilot, accédez aux parametres, puis "Model Matching (MSFS)" et enfin "Custom Rules" dans le sous-menu déployable grace au "+"





2- Cliquez sur "Add Custom Rule Set(s)..." et localisez le fichier .vmr que vous venez de télécharger (l'emplacement de ce fichier n'a pas d'importance), une fois le fichier .vmr séléctionné cliquez sur "Ouvrir"



3- La livrée et ses règles de model matching sont maintenant installées dans votre client vPilot.

4- Répètez avec les étapes 2 et 3 pour les autres avions si besoin.

5-Une fois tous les fichiers .vmr entrés dans vPilot, toujours dans le menu "Custom Rules", séléctionnez le fichier .vmr défault du Flying Club et placez le en bas de liste avec le bouton "Move Down"



6- Si les fichiers .vmr ont un ordre similaire a celui-ci :
French_Flying_Club-PMDG-XXXXXXXX.vmr
French_Flying_Club-Default-XXXXXXXX.vmr



Les règles de model matching devrait fonctionner correctement. Un test en Observer vous le confirmera !

PACK AIG

Si les autres avions n'ont pas de livrées vous pouvez suivre les étapes suivantes pour installer le pack AIG.
Le pack AIG est téléchargeable via ce lien : ici
Ce pack contient les livrées et modèles d'avions des principales compagnies.
Pour l'installer :

1- Extraire le contenu du fichier .zip vers un dossier AIG (le nom de ce dossier n'as pas d'importance), vous pouvez utiliser la fonction "Extraire vers [nom du dossier]"

2- Placer ce dossier dans le dossier "Community" et localisez le fichier .vmr

3- Dans le client vPilot, accédez aux paramètres, puis "Model Matching (MSFS)" et enfin "Custom Rules" dans le sous-menu déployable grâce au "+"

4- Cliquez sur "Add Custom Rule Set(s)..." et localisez le fichier .vmr, une fois le fichier .vmr séléctionné cliquez sur "Ouvrir".

5- Une fois le fichier .vmr présent dans liste de vPilot, faites le descendre tout en bas avec le bouton "Move Down"

6- Le pack AIG et ses règles de model matching sont maintenant installées dans votre client vPilot.
